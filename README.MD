## COVID-19 Info ##

Simple Javafx Application for view updates from the COVID-19 deseases

Library in this project :
1. Retrofit HTTPClient
2. RAPIDAPI for Covid-19 Data by 
[Gramzivi](https://rapidapi.com/Gramzivi/api/covid-19-data)
